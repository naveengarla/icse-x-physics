## Topics Covered

### 3.1 Introduction to Machines
- A **machine** is a device that makes work easier — it transforms energy and multiplies force or speed or changes direction of force
- A machine cannot create energy; it can only transfer or transform it
- **Input**: effort force applied; **Output**: load lifted/moved

### 3.2 Key Terms and Efficiency
| Term | Definition | Formula |
|---|---|---|
| Mechanical Advantage (MA) | Ratio of load to effort | `MA = Load (L) / Effort (E)` |
| Velocity Ratio (VR) | Ratio of distance moved by effort to distance moved by load | `VR = dE / dL` |
| Efficiency (η) | Ratio of output work to input work | `η = (MA/VR) × 100%` |

- For an **ideal machine**: MA = VR, efficiency = 100%
- Real machines: MA < VR, efficiency < 100% (due to friction)
- `η = (Load × dL) / (Effort × dE) × 100%`

### 3.3 Levers
- A **lever** consists of a rigid rod or bar pivoted at a fixed point called the **fulcrum (F)**
- Three parts: Load (L), Effort (E), Fulcrum (F)

#### Classes of Levers

| Class | Position | MA | Example |
|---|---|---|---|
| **First Class** | Fulcrum between Load and Effort | Can be > 1, = 1, or < 1 | See-saw, scissors, crowbar, beam balance |
| **Second Class** | Load between Fulcrum and Effort | Always > 1 | Wheelbarrow, nutcracker, bottle opener |
| **Third Class** | Effort between Fulcrum and Load | Always < 1 | Tweezers, forearm, sugar tongs |

- MA of lever = `Effort arm / Load arm`
- VR of lever = `Effort arm / Load arm` (same as MA for ideal lever)

### 3.4 Pulley Systems
A **pulley** is a wheel with a groove used to change direction or magnitude of a force.

#### Single Fixed Pulley
- Fulcrum at centre
- MA = 1 (no mechanical advantage in force)
- VR = 1
- Efficiency = 100% (ideal)
- Use: changes **direction** of effort (e.g., raising a flag)

#### Single Movable Pulley
- Load is attached to movable pulley
- MA = 2 (ideal); MA < 2 (real, due to weight of pulley and friction)
- VR = 2
- Use: multiplies force by 2

#### Block and Tackle (Combination Pulley)
- Combination of fixed and movable pulleys
- VR = total number of pulleys (= number of rope segments supporting movable block)
- MA (ideal) = VR
- As number of pulleys increases, MA increases but efficiency decreases (more friction)

| System | VR | Ideal MA |
|---|---|---|
| 1 fixed + 1 movable | 2 | 2 |
| 2 fixed + 2 movable | 4 | 4 |
| n pulleys (block & tackle) | n | n |

| Quantity | Formula |
|---|---|
| Mechanical Advantage | `MA = L/E` |
| Velocity Ratio | `VR = dE/dL` |
| Efficiency | `η = (MA/VR) × 100%` |
| Lever MA | `MA = Effort arm / Load arm` |
| Pulley VR | `VR = number of rope sections supporting load` |

## Important Terms

- **Fulcrum**: The fixed point about which a lever rotates
- **Effort Arm**: Perpendicular distance from fulcrum to line of action of effort
- **Load Arm**: Perpendicular distance from fulcrum to line of action of load
- **Mechanical Advantage**: The factor by which a machine multiplies the applied force
- **Efficiency**: Percentage of input energy converted to useful output energy

---

## Human Body as a Machine
- Forearm: 3rd class lever (elbow = fulcrum, bicep = effort, load in hand)
- Head nodding: 1st class lever (neck = fulcrum)
- Standing on toes: 2nd class lever (toe = fulcrum, calf muscle = effort, body weight = load)
